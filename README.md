# Power BI Capstone Project – Consumer Electronics Sales Analysis

## Project Overview
This is an end-to-end Power BI project focused on analyzing sales performance for a consumer electronics company operating across the United States. The project demonstrates the complete data analytics lifecycle, from data ingestion and transformation to modeling, visualization, and reporting.

## Business Objective
The objective of this project is to generate actionable insights that support business decision-making, including revenue trends, customer behavior, product performance, employee efficiency, and regional profitability.

## Dataset Description
The analysis is based on multiple datasets covering the period 2018–2022:
- Customer profile data
- Product details
- Sales transactions (combined from multiple files)
- Sales employee data
- Order and transaction date data (combined from multiple files)

## Tools and Technologies
- Power BI Desktop
- Power Query for data transformation
- Star schema data modeling
- DAX for calculated columns and measures
- Interactive dashboards and slicers
- Mobile report layout
- PDF export for print-ready reporting

## Key Insights Delivered
- Year-wise total revenue and customer count
- Total employees, states served, products, and categories
- Top and bottom products, categories, employees, and states by revenue
- Employee performance by quantity sold and revenue after discount
- Quarterly revenue comparison across multiple years
- Key influencer analysis to identify drivers of revenue

## Deliverables
- Interactive Power BI dashboard
- Mobile-optimized report layout
- Exported PDF version for sharing and printing
